.. _cmd:

ChainerUI command manual
========================

.. _cmd_server:

Server
------

::

  $ chainerui server

* ``--host``: (optional)
* ``--port``: (optional)
* ``--debug``: (optional)

.. _cmd_database:

Database
--------

::

  $ chainerui db create

::

  $ chainerui db upgrade

::

  $ chainerui db drop

.. _cmd_project:

Project
-------

::

  $ chainerui project create

* ``-d``: (required)
* ``-n``: (optional)
